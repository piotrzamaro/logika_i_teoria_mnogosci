Klasyczny Rachunek Zdań


Rachunek zdań jest podstawowym działem logiki matematycznej, który zajmuje się badaniem związków między zdaniami, ich prawdziwością i fałszywością oraz operacjami, które można na nich wykonywać.

Zdanie logiczne

Zdanie logiczne to wyrażenie, które ma jednoznaczną wartość logiczną – jest albo prawdziwe (P), albo fałszywe (F). Przykłady zdań logicznych to: - “2 + 2 = 4” – jest prawdziwe. - “Koty potrafią latać” – jest fałszywe.

Zdania logiczne mogą być zarówno proste, jak i złożone. Zdania złożone powstają przez połączenie zdań prostych za pomocą spójników logicznych.

Spójniki logiczne

Spójniki logiczne to operatory, które łączą zdania logiczne, tworząc złożone formuły. Oto najważniejsze spójniki klasycznego rachunku zdań:

  1. Negacja (¬) – zmienia wartość logiczną zdania na przeciwną:
    • Jeśli ( A ) jest prawdziwe, to ( ¬A ) jest fałszywe.
    • Jeśli ( A ) jest fałszywe, to ( ¬A ) jest prawdziwe.
  2. Koniunkcja (∧) – “i”, zdanie złożone prawdziwe tylko wtedy, gdy oba zdania składowe są prawdziwe:
    • ( A ∧ B ) jest prawdziwe, jeśli zarówno ( A ), jak i ( B ) są prawdziwe.
  3. Alternatywa (∨) – “lub”, zdanie złożone jest prawdziwe, gdy co najmniej jedno ze zdań składowych jest prawdziwe:
    • ( A ∨ B ) jest prawdziwe, jeśli ( A ), ( B ) lub oba są prawdziwe.
  4. Implikacja (→) – “jeśli… to…”, prawdziwe, gdy jeśli pierwsze zdanie jest prawdziwe, to drugie musi być prawdziwe:
    • ( A → B ) jest fałszywe tylko wtedy, gdy ( A ) jest prawdziwe, a ( B ) fałszywe.
  5. Równoważność (↔︎) – “wtedy i tylko wtedy”, prawdziwe, gdy oba zdania mają taką samą wartość logiczną:
    • ( A ↔︎ B ) jest prawdziwe, gdy ( A ) i ( B ) są jednocześnie prawdziwe lub jednocześnie fałszywe.

Formuły logiczne

Formuła logiczna to wyrażenie zbudowane ze zdań prostych i spójników logicznych. Każda formuła ma przypisaną wartość logiczną, która zależy od wartości logicznych zdań prostych oraz zastosowanych spójników.

Przykład formuły logicznej: \[ (A ∧ B) → C \]

Tautologie

Tautologia to formuła logiczna, która jest zawsze prawdziwa, niezależnie od wartości logicznych zdań, które ją tworzą. Przykładem tautologii jest formuła: \[ A ∨ ¬A \] Znaczy to, że zdanie ( A ) jest albo prawdziwe, albo jego zaprzeczenie jest prawdziwe – co zawsze będzie prawdą.

Ważniejsze prawa klasycznego rachunku zdań

  1. Prawo podwójnego zaprzeczenia: \[ ¬(¬A) ≡ A \] Dwukrotne zaprzeczenie zdania jest równoważne samemu zdaniu.

  2. Prawo rozdzielności: \[ A ∧ (B ∨ C) ≡ (A ∧ B) ∨ (A ∧ C) \]

  3. Prawo De Morgana:

    • ( ¬(A ∧ B) ≡ (¬A ∨ ¬B) )
    • ( ¬(A ∨ B) ≡ (¬A ∧ ¬B) )

  4. Prawo kontrapozycji: \[ (A → B) ≡ (¬B → ¬A) \]

  5. Prawo identyczności: \[ A ∧ A ≡ A \]

  6. Prawo wykluczającego środka: \[ A ∨ ¬A \]

Postać normalna formuły (APN i KPN)

Postać normalna alternatywna (APN) oraz kanoniczna postać normalna (KPN) to sposoby zapisu formuł w logice, które upraszczają analizę formuł.

  • Alternatywna postać normalna (APN) – formuła jest w postaci alternatywy koniunkcji.
    • Przykład: ( (A ∧ B) ∨ (¬A ∧ C) )
  • Koniunktywna postać normalna (KPN) – formuła jest w postaci koniunkcji alternatyw.
    • Przykład: ( (A ∨ B) ∧ (¬A ∨ C) )

Obie postacie są pomocne w uproszczeniu formuł logicznych oraz przy budowie systemów informatycznych.

Metoda tablic analitycznych

Metoda tablic analitycznych to graficzna metoda weryfikacji formuł logicznych, która umożliwia rozstrzygnięcie, czy dana formuła jest tautologią, spełnialna, czy sprzeczna. Działa poprzez rozbicie formuły na wszystkie możliwe przypadki wartości logicznych, a następnie weryfikowanie, czy można uzyskać wszystkie możliwe wyniki.

Kroki w metodzie tablic analitycznych:

  1. Rozbicie formuły na alternatywy i koniunkcje.
  2. Podział na gałęzie w drzewie dla różnych możliwych wartości logicznych.
  3. Ocena, czy każda gałąź prowadzi do sprzeczności (oznacza, że formuła nie może być spełniona) lub do wartości prawdziwej (oznacza, że formuła jest spełnialna lub tautologiczna).

Podsumowując, rachunek zdań to kluczowa część logiki matematycznej, która pozwala analizować związki między zdaniami przy użyciu spójników logicznych. Dzięki takim narzędziom jak tabele prawdy, prawa logiki i metoda tablic analitycznych można sprawdzać poprawność wniosków logicznych i analizować złożone wyrażenia logiczne.
